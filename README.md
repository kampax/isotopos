# Campaña Isótopos 2026

Este sitio web contiene la documentación fotográfica y mapas interactivos de los puntos de muestreo.

## Estructura
- `index.html` - Página principal con índice de todos los sitios
- `mapa_isotopos.html` - Mapa general con todos los puntos
- `I01.html` a `I12.html` - Páginas individuales por sitio
- `Fotos/` - Imágenes asociadas a cada sitio

## Hosting
Este sitio es estático y puede ser alojado en:
- GitHub Pages
- Netlify
- Vercel
- Cualquier servidor web estático

Para visualizar localmente, simplemente abre `index.html` en tu navegador.
